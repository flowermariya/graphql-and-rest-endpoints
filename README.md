## Bookshelf Management

## Description

This API allows users to manage a virtual bookshelf. Users can add, update, delete, and retrieve books. The API also supports user authentication and management

## Installation

$ npm install


## Running the app

# development
$ npm run start

# watch mode
$ npm run start:dev

```
